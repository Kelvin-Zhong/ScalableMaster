# ScalableMaster

##### Project: Gatheracty
 We intend to create an website that can help merchants to publish their activities. At the same time, we also intend to let users to show their interest on different categories of activities so that they will not miss any exciting activities nearby. Gatheracty stands for Gather Activities.
